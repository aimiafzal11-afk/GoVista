# GoVista
Project Title: Explore Pakistan – Travel Website
==============================================

Project Description
-------------------
This project is a responsive travel website designed to showcase the
natural beauty, cultural heritage, and famous tourist destinations of
Pakistan. The website aims to provide users with an engaging visual
experience while maintaining a clean and professional interface.

The project demonstrates front-end web development skills using modern
web technologies and focuses on user experience, accessibility, and
aesthetic design.

----------------------------------------------
Objectives
----------------------------------------------
• To design an informative and visually appealing travel website
• To practice responsive web design principles
• To implement dark and light mode functionality
• To organize destinations into meaningful sections
• To enhance user interaction using modals and hover effects

----------------------------------------------
Project Features
----------------------------------------------
• Home page with hero section and introduction
• Places page with categorized destinations
• Image cards with modal previews
• Uniform image sizing across sections
• Dark / Light mode toggle using moon–sun icon
• Responsive layout for mobile and desktop devices
• Clean typography and accent-based color scheme

----------------------------------------------
Pages Included
----------------------------------------------
• index.html        – Home Page
• places.html       – Tourist Destinations
• about.html        – Project Overview

----------------------------------------------
Technologies Used
----------------------------------------------
• HTML5
• CSS3
• Bootstrap 5
• JavaScript
• Font Awesome Icons

----------------------------------------------
How to Run the Project
----------------------------------------------
1. Download or extract the project folder
2. Open the file "index.html" in any web browser
3. Navigate through pages using the navigation bar
4. Click on images to view them in modal windows
5. Use the toggle button to switch between dark and light mode
