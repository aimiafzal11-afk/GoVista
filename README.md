# GoVista 🌍

### Explore Pakistan – Travel & Tourism Website

## Project Description

GoVista is a responsive travel and tourism website designed to showcase the natural beauty, cultural heritage, and popular tourist destinations of Pakistan. The project provides users with an engaging and visually appealing experience while maintaining a clean, modern, and user-friendly interface.

Built using modern front-end technologies, the website focuses on responsive design, accessibility, and intuitive navigation. It serves as a practical demonstration of web development skills, UI/UX principles, and interactive user experiences.

## Features

- Responsive design optimized for desktop, tablet, and mobile devices
- Attractive hero section with engaging travel-themed content
- Categorized tourist destinations from across Pakistan
- Interactive image cards with modal preview functionality
- Dark and Light mode theme switching
- Consistent image sizing and layout structure
- Modern typography and clean user interface
- Smooth navigation across website pages
- User-friendly and accessible design principles

## How to Run the Project

1. Download or clone the repository.
2. Extract the project folder (if downloaded as a ZIP file).
3. Open the project directory.
4. Launch `index.html` in any modern web browser.
5. Navigate through the website using the navigation menu.
6. Click destination images to view enlarged previews in modal windows.
7. Use the theme toggle button to switch between Dark and Light modes.
